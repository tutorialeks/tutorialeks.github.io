### APPENDIX D: TRACKING SPREADSHEET FORMULAS REFERENCE

#### Google Sheets Formula Library

**Cost Per Lead (CPL) Calculation:**
```
=IF(D2>0, C2/D2, 0)
```
Where: C2 = Total Spent, D2 = Form Submissions
Result: Shows cost to acquire each lead

**Conversion Rate:**
```
=IF(E2>0, (G2/E2)*100, 0)
```
Where: E2 = Clicks, G2 = Form Submissions
Result: Percentage of clicks that converted to leads

**ROI (Return on Investment):**
```
=IF(C2>0, ((H2-C2)/C2)*100, 0)
```
Where: H2 = Revenue, C2 = Total Spent
Result: Percentage return on ad spend

**Remaining Budget:**
```
=B2-C2
```
Where: B2 = Budgeted Amount, C2 = Amount Spent
Result: How much budget is left

**Weekly Summary Auto-Sum:**
```
=SUMIFS('Daily Metrics'!I:I, 'Daily Metrics'!A:A, ">="&B2, 'Daily Metrics'!A:A, "<="&C2)
```
Where: 'Daily Metrics'!I:I = Cost column, B2 = Week start date, C2 = Week end date
Result: Automatically sums all costs within date range

**Variant Performance Comparison:**
```
=COUNTIFS('Daily Metrics'!H:H, "WHITE", 'Daily Metrics'!G:G, ">0") / COUNTIFS('Daily Metrics'!H:H, "WHITE")
```
Result: Conversion rate for White theme variant specifically
