# 100 Facebook Text-Over-Image Ad Ideas for TutoriAleks
## Organized by Psychological Trigger & Campaign Phase


# USAGE INSTRUCTIONS

### How to Use These 100 Ad Ideas:

**Phase 1 (Week 1-2): Foundation**
- Use ads from: Category 1 (Pain), Category 2 (Solution), Category 6 (Education)
- Goal: Build awareness that individual tutoring is the solution

**Phase 2 (Week 2-3): Trust Building**
- Use ads from: Category 3 (Social Proof), Category 5 (Value Prop)
- Goal: Show credibility and unique value

**Phase 3 (Week 3-4): Urgency + Conversion**
- Use ads from: Category 4 (Urgency), Category 7 (Direct CTA)
- Goal: Drive immediate action

**Phase 4 (Ongoing): Engagement**
- Use ads from: Category 8 (Humor/Relatable)
- Goal: Build community and keep audience warm

### Production Tips:

**Design Tools:**
- Canva: Search "Facebook Ad" template (1080x1080 or 1200x628)
- Remove template text
- Add your background image
- Overlay text in bold, readable font (Inter Black, Montserrat Bold)

**Typography Rules:**
- Font size: 60-80pt for headline
- Color: High contrast (white text on dark image, or vice versa)
- Add subtle shadow/stroke for readability
- Max 3 lines of text for mobile readability

**Background Image Sources:**
- Unsplash: Search "student studying", "laptop coding", "frustrated", "happy learning"
- Pexels: Similar search terms
- AI Generate: Use Midjourney/DALL-E with prompts from descriptions above

**A/B Testing:**
- Test 3-5 variations per week
- Same image, different text
- Same text, different image
- Track which performs best

### Customization:

For each ad, you can create variations:
1. Change тон (formal ↔ informal)
2. Change urgency level (calm ↔ panic)
3. Change audience (students ↔ professionals)
4. Change language (Bulgarian ↔ English for future)
